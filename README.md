This project analyzes engineering student data to identify factors that influence employability and placement readiness. Using Python for data processing and Power BI for visualization, the system evaluates student performance based on academic scores, technical skills, internships, hackathon participation, and other key attributes.

A custom “Placed” indicator column is generated by comparing each student's overall performance with the dataset average across multiple parameters such as tenth percentage, twelfth percentage, CGPA, backlogs, internships, and hackathon participation. Students meeting or exceeding the average criteria are categorized as placed, while others are marked as not placed.

The project provides interactive dashboards and visual insights that help identify skill gaps, analyze placement trends, and support data-driven decisions for improving student employability.
